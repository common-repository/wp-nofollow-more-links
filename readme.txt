== Description == 

Adds the "nofollow" rel attribute to the read more links. No complicated setup. Just activate and everything will work automatically!

Plugin by <a href="http://tutskid.com/">TutsKid | WordPress Tutorials, Themes, Plugins, and More!</a>.

== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's all, everything will work automatically
